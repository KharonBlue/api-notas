# API NOTAS
API REST para la gestión de materias de un colegio con sus respectivos profesores.

-La mayoria de la aplicación esta creada y configurada en español.
-Test configurados para ejecutarse desde en una base de datos en memoria(H2).
-Los paquetes estan nombrados en plurar pero las clases en singular.
-Log configurado para guardarse en el paquete "logs".


### CORRER LOCALMENTE

En este proyecto ejecutar:
```
mvn spring-boot:run
```
Tener en cuenta la configuración de la base de datos.

### SWAGGER
